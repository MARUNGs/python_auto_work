## 작업순서
1. 작업할 엑셀 파일을 업로드한다.
2. 희망e음코드를 작성한다.
3. 각 첨부파일 파일명, 파일경로, 희망e음코드가 존재하면 엑셀 테이블을 그린다.
    - 엑셀파일의 각 행을 돌아 행별 열값을 list로 삽입한다.
    - list로 저장한 데이터를 최종 list에 삽입한다.
    - QTableWidget 객체를 호출하여, 최종 list에 담긴 데이터를 설정하여 자동화 작업 전의 데이터를 사용자에게 보여준다.

4. 윈도우에 실행중인 프로그램 이름이 '사회복지시설정보시스템(1W)'가 포함되는 프로그램을 찾아서 호출한다.
(이 때, 이미 프로그램이 실행되고 있는 상태에서 결의서/전표 간편입력 화면을 띄우고 있는 상태여야 한다.)
5. 신규 이미지와 동일한 버튼을 찾아서 가운데를 클릭한다.
6. 각 항목에 대한 값을 리스트에 보관한 값대로 세팅한다.
